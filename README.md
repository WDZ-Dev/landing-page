# WDZ Solutions

Landing page for [WDZ Solutions](https://wdz-landing.pages.dev) — AI agents, custom software, and applications for businesses.

## Stack

- React + TypeScript + Vite
- Firebase Firestore (contact form + quote submissions)
- Cloudflare Pages (hosting)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
